使い方
1．ffmpeg-*.*.*-win64-static.zip をダウンロードしてきて展開する
	例えば https://ffmpeg.zeranoe.com/builds/ 
	Version は上から 2 個目の項目（Release builds）がおすすめ
2．<展開先フォルダ>\ffmpeg-*.*.*-win64-static\bin にパスを通す
3．変換したいファイルをバッチファイルまでドラッグアンドドロップする
4．処理が終わったらできあがり

バッチファイル名について
ffmpeg_<コーデック名>_<インターレース解除の有無>_<使用技術>
・コーデック名:ここに書かれているコーデックへ変換します。コンテナはすべて MP4 です。
  avc はビデオ H.264 AVC、オーディオ AAC のことです。通常はこちらを使うことをおすすめします。
  hevc はビデオ H.265 HEVC、オーディオ AAC のことです。圧縮率が高いです。
・インターレース解除の有無
  deinterlace と書いてあるものはインターレース解除が入っています。
  元データがインターレースの場合は deinterlace または deinterlace60p と書いてあるものを使ってください。
　急いでいない場合は 60p がおすすめです。
  元データがプログレッシブの場合は何も書いていないものを使ってください。
・使用技術
　Intel は Intel GPU に内蔵の HW エンコーダーを使用して処理をします。非常に高速です。
　Intel Core i3-8130（Kaby Lake Refresh/UHD 620）で動作確認をしました。
　SW は CPU で処理します。

処理内容
　ffmpeg（オープンソースのマルチメディアライブラリ）を使って、エンコード処理をするだけのもの。

補足1:品質指定
・Intel の場合
　-global_quality <ここに2桁の自然数>
　2桁で決めるだけでビットレートを良い感じに調整してくれます。
　数字が「小さいほど」高品質です。
　だいたい 15～25 の間で調整すれば良いと思います。
・SW の場合
　-crf <ここに2桁の自然数>
　2桁で決めるだけでビットレートを良い感じに調整してくれます。
　数字が「小さいほど」高品質です。
　ffmpeg におけるデフォルト値は 23 です。

補足2:インターレース解除の設定
 bwdif=A:B:C
・A:モード指定。0で通常（30fps）、1で方フィールド補完（60fps）
・B:フィールドタイプ指定。-1で自動、0でtff、1でbff
・C:どのフレームを処理するか。0ですべて、1でインターレースフラグがあるもののみ。
